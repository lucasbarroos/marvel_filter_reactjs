# Project Title

A simple ReactJs project to consume the data of the Marvel Api

## Getting Started

To run this application, follow the next steps

### Prerequisites

You have to install this following technologies

```
Node
Npm
ReactJS
```

### Installing

Just run this following commands

```
yarn
```

```
yarn start
```
